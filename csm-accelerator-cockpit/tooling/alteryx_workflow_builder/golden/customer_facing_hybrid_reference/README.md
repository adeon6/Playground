# Customer-Facing Hybrid Workflow Reference

This is the primary golden reference for customer-presentable generated workflows.

It demonstrates the merged beautification direction:

- colored contextual containers around every real tool group
- a dark customer-facing title banner above the tool body
- clean left-to-right lanes and hub placement
- concise native tool annotations
- a bottom documentation shelf
- an optional side tip panel
- right-side output anchoring

Use `10_REFERENCE_customer_facing_hybrid.yxmd` as the visual standard and `10_REFERENCE_customer_facing_hybrid.png` as the quick preview.

This reference is about canvas structure and presentation quality. Adapt the business logic to the customer use case, but preserve the visual grammar unless the user explicitly asks for another style.
