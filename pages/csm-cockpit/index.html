Local Python App / V5.8

Accelerator Cockpit

Guided intake for accelerator discovery, evidence review, SOP approval, and workflow handoff preparation.

V5.8 Release package

Run Locally

Download the ZIP, extract the app folder, then double-click:

csm-accelerator-cockpit-v5.8\Start Accelerator Cockpit.bat

The launcher creates a local environment, starts the app, and opens the browser. Customer transcripts stay on the machine.

Download V5.8 Standard ZIP Download V5.8 + Global Skills ZIP Open GitHub Folder

Use the standard ZIP for self-contained cockpit projects. Use the global-skills ZIP only when you also want to update Codex's global Alteryx workflow skills on that machine.
