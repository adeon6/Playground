CSM Accelerator Cockpit

A guided discovery cockpit for CSMs. GitHub Pages hosts this landing page and the ZIP, while the transcript app runs locally so customer content stays on the machine.

How Jon Runs It

Download the ZIP, extract the `csm-accelerator-cockpit-v4` folder, then double-click:

csm-accelerator-cockpit-v4\Start CSM Cockpit.bat

The launcher creates a local environment, starts the app, and opens the browser.

What Powers It

V4.1 uses an explainable local rules engine. It starts with transcript review, reads DOCX, Markdown, or text transcripts, maps evidence to Jon's section-level discovery model, creates the docs/status/workflows project structure, and gates workflow handoff behind human approval.

The bundled process pack includes Jon's interview script, SOP templates, sequencer config, and the Large-only workflow-build prompt pack. V4.1 also bundles the Alteryx workflow-builder toolkit and beautification rules, then generates a project-specific Codex prompt and helper script when the SOP gate is ready. The app now detects packaged Windows Codex installs more reliably and keeps Designer-safe TextBox styling rules in the bundled workflow tooling.

No customer transcript is uploaded. The sample transcript is sanitized.
